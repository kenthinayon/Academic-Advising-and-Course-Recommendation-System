# College logos

Drop college logo image files in this folder so the UI can load them.

## Expected files

These filenames are used by the Admin “Department Summary” cards.

Image format: you can use `.png`, `.jpg`, or `.jpeg` (the UI will try png → jpg → jpeg automatically).

- `accountancy.png`
- `business-administration.png`
- `computer-studies.png`
- `engineering-technology.png`
- `nursing.png`
- `criminal-justice-education.png`
- `teacher-education.png`
- `arts-sciences.png`

If you use a different file name or extension, update the `logo` field in `resources/js/components/AdminDashboard.js`.
